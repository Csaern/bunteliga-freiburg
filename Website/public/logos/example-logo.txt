# Beispiel-Logo Platzhalter

Dies ist ein Platzhalter für Team-Logos.

## So verwenden Sie dieses Verzeichnis:

1. **Logo-Dateien hochladen:**
   - Laden Sie Ihre Team-Logos in dieses Verzeichnis hoch
   - Verwenden Sie aussagekräftige Dateinamen wie:
     - fc-freiburg.png
     - sc-freiburg.jpg
     - team-123.gif

2. **Logo-URLs verwenden:**
   - Nach dem Upload können Sie die Logos über URLs wie diese verwenden:
   - https://ihre-domain.de/logos/fc-freiburg.png
   - https://ihre-domain.de/logos/team-123.jpg

3. **In der Anwendung zuweisen:**
   - Gehen Sie zum AdminBoard → Teams & Logos
   - Wählen Sie "Plesk File Manager (URL-basiert)"
   - Fügen Sie die Logo-URL für das entsprechende Team ein

## Empfohlene Logo-Spezifikationen:

- **Format:** PNG (mit Transparenz) oder JPG
- **Größe:** 200x200 Pixel (quadratisch)
- **Dateigröße:** Maximal 500KB
- **Auflösung:** 72-150 DPI

## Unterstützte Formate:

- PNG (empfohlen für Logos mit Transparenz)
- JPG (empfohlen für Fotos)
- GIF (für animierte Logos)
- SVG (für skalierbare Vektorgrafiken)
- WebP (moderne Komprimierung)

## Beispiel-Dateistruktur:

```
/logos/
├── fc-freiburg.png
├── sc-freiburg.jpg
├── sv-freiburg.gif
├── team-123.png
├── verein-logo.svg
└── example-logo.txt (diese Datei)
```

Löschen Sie diese Datei nach dem Upload Ihrer ersten Logos.
